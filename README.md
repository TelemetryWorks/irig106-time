# irig106-time

**Note:** This crate is currently a placeholder.

Nanosecond-precision time handling for IRIG 106. This library is being developed to provide strictly defined, standalone logic for interpreting, converting, and syncing the complex time formats embedded within aerospace telemetry payloads.

We have secured this name on `crates.io` while we finalize the initial version.

Please check the [active GitHub repository](https://github.com/TelemetryWorks/irig106-time) to see our current progress.
